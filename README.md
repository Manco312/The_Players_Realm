# 🎮 The Players Realm – Frontend

Aplicación web desarrollada con **Vue.js** para la gestión y visualización de videojuegos, estudios y reseñas dentro de la plataforma **The Players Realm**.

El sistema permite visualizar información de videojuegos, incluyendo género, precio, ventas, director y estudio desarrollador.

---

# 🚀 Tecnologías utilizadas

* **Vue.js**
* **TypeScript**
* **Vite**
* **Pinia** (manejo de estado)
* **ESLint** (análisis de código)
* **Prettier** (formateador de código)

---

# 📦 Instalación del proyecto

1️⃣ Clonar el repositorio:

```bash
git clone https://github.com/Manco312/The_Players_Realm.git
```

2️⃣ Entrar al proyecto:

```bash
cd frontend
```

3️⃣ Instalar dependencias:

```bash
npm install
```

---

# ▶️ Ejecutar el proyecto

Para ejecutar la aplicación en modo desarrollo:

```bash
npm run dev
```

Luego abre el navegador en:

```
http://localhost:5173
```

---

# 🧭 Ruta principal de la aplicación

La ruta principal del sistema es:

```
/
```

Desde esta ruta se cargan los componentes principales de la aplicación y se visualiza la información de videojuegos.

---

# 📁 Estructura básica del proyecto

```
THEPLAYERSREALM
│
└── frontend
    │
    ├── public/               # Archivos públicos accesibles desde el navegador
    │
    ├── src/
    │   ├── assets/           # Recursos estáticos
    │   ├── components/       # Componentes reutilizables de Vue
    │   ├── constants/        # Constantes globales del proyecto
    │   ├── dtos/             # Data Transfer Objects
    │   ├── interfaces/       # Interfaces TypeScript
    │   ├── router/           # Configuración de rutas de Vue Router
    │   ├── seeders/          # Datos iniciales del sistema
    │   ├── services/         # Lógica de negocio y servicios
    │   ├── stores/           # Manejo de estado con Pinia
    │   ├── types/            # Tipos personalizados de TypeScript
    │   ├── utils/            # Funciones utilitarias
    │   ├── views/            # Vistas principales de la aplicación
    │   │
    │   ├── App.vue           # Componente raíz de la aplicación
    │   ├── main.ts           # Punto de entrada de la aplicación
    │   └── PiniaConfig.ts    # Configuración de Pinia
```
